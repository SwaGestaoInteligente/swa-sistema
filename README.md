# 🦁 SWA Gestão Inteligente

Sistema de gestão multi-organização modular e mobile-first.

## 🎯 Para Quem

- 🏢 **Condomínios** - Gestão completa de moradores, áreas, vistorias
- 🏥 **Clínicas** - Pacientes, agendamentos, equipamentos
- 🏭 **Empresas** - Colaboradores, chamados, reservas
- ⛪ **Igrejas** - Membros, dízimos, eventos

## ⚡ Módulos Disponíveis

### ✅ Campo/Vistoria
- Checklists customizáveis por categoria
- Fotos inline com preview
- Cálculo automático de risco
- Relatórios PDF profissionais
- Envio por email automático

### 💰 Financeiro (em desenvolvimento)
- Plano de contas hierárquico
- Lançamentos (pagar/receber)
- Fluxo de caixa e DRE
- Inadimplência

### 👥 Pessoas
- Gestão contextual por tipo de organização
- Vínculos por período
- Histórico completo

### 📅 Agendamentos (em desenvolvimento)
- Recursos (salas, consultórios, equipamentos)
- Calendário inteligente
- Confirmação automática

## 🚀 Stack Tecnológico

- **Backend:** Laravel 12
- **Frontend:** Blade + Vite + Alpine.js
- **Database:** PostgreSQL 16
- **Deploy:** Fly.io + GitHub Actions
- **PWA:** Service Worker + Offline-first

## 📱 Mobile-First

- ✅ PWA instalável (iOS + Android)
- ✅ Funciona offline
- ✅ Sincronização automática

## 🔒 Segurança

- ✅ Multi-tenant isolado
- ✅ Auditoria completa
- ✅ Backup automático
- ✅ LGPD-ready
